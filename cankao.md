🎨 提示词生成逻辑
三个核心变量（由 AI 分析提取）
变量名含义示例style_type整体风格与制作级别"专业日式TV动画赛璐璐制作级别"surface_texture表面质感渲染工艺"平滑细腻的赛璐璐色块填充与锐利清晰的轮廓描线工艺"proportion人体比例体系"标准日式动画八头身比例"
两种布局模式
1.宽幅布局 (panoramic)：左一（头部特写）+ 右三（正面/侧面/背面全身）
2.6宫格布局 (grid2x2)：3列×2行，6个局部特写（侧颜、顶部、后颈、胸肩、腰饰、鞋靴）
三种输出格式
- 中文提示词：用于中文生图模型
- 英文提示词：用于英文生图模型
- JSON 格式：结构化输出，方便程序调用
--------------------------------------------------------------------------------
🔌 Vision LLM 调用逻辑 (vision-analyzer.ts)
调用流程：
1.将用户上传的图片压缩至 1024px 以内，转为 base64
2.根据选择的服务商，构造不同的请求格式：
- OpenAI 兼容格式：/chat/completions 端点，带 image_url 多模态内容
- Google Gemini 格式：/models/{model}:generateContent?key={key} 端点
3.发送系统提示词 + 图片，要求 AI 返回 JSON 格式的三个变量（含中英文）
4.解析并校验返回结果
系统提示词设计要点：
- 要求 AI 只提取「大方向级别的极简防跑偏文字锚点」
- 禁止描述具体视觉细节、环境光照、镜头景深等
- 对 2.5D 混合风格，要求翻译为「工业级三维中间层风格化物理渲染」等高级术语

这是一个设计非常专业的工具，核心思路是：用 Vision LLM 提取画风锚点 → 填入精心设计的提示词模板 → 生成可直接用于生图的高质量提示词，特别适合为视频生成模型（如 Seedance）提供「视觉真值（Ground Truth）」角色基准资产。
